# 🌸 Iris Flower Classification using KNN

## Overview

This project implements a Machine Learning classification model using the **K-Nearest Neighbors (KNN)** algorithm to predict the species of an Iris flower based on its sepal and petal measurements.

Developed as part of the **DecodeLabs Artificial Intelligence Internship**, this project demonstrates the fundamentals of supervised learning, data preprocessing, model training, and evaluation.

## Features

* Iris Flower Species Classification
* Data Preprocessing and Analysis
* Feature Scaling using StandardScaler
* Train-Test Data Splitting
* KNN Model Training
* Accuracy Evaluation
* Confusion Matrix Generation
* Classification Report
* User Input-Based Flower Prediction

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Workflow

Iris Dataset → Data Preprocessing → Feature Scaling → Train-Test Split → KNN Model Training → Prediction → Model Evaluation

## Learning Outcomes

* Understanding Supervised Learning
* Working with Real-World Datasets
* Data Preprocessing Techniques
* Feature Scaling
* K-Nearest Neighbors (KNN) Algorithm
* Model Training and Testing
* Performance Evaluation using Accuracy, Precision, Recall, and F1-Score
