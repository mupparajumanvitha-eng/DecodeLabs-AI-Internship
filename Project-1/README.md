# 🤖 DecodeLabs Internship Project

## 📌 Project Overview

This project is a Rule-Based AI Chatbot developed using Python as part of my Artificial Intelligence Internship at DecodeLabs.

The chatbot simulates basic human conversation by responding to predefined user inputs using logical rules and decision-making techniques. It can greet users, tell jokes, provide motivational quotes, display the current date and time, and respond to simple mood-based interactions. This project helped me understand the fundamentals of conversational AI and how chatbots work using rule-based systems.

---

## ✨ Features

* Interactive text-based chatbot
* Greeting responses
* Random jokes
* Motivational quotes
* Compliment generator
* Current date and time
* Mood-based responses
* Simple help menu
* Exit command
